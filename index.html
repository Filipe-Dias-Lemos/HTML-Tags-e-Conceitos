Estrutura básica

Voltar

Uma página HTML básica geralmente começa com a declaração do tipo de documento, indicada pela tag <!DOCTYPE html>. Em seguida, o elemento <html> é usado para envolver todo o conteúdo da página. Dentro do elemento <html>, são geralmente encontrados dois principais blocos: o <head> e o <body>. O <head> contém metadados sobre o documento, como título, codificação de caracteres, folhas de estilo CSS e scripts JavaScript, enquanto o <body> contém o conteúdo visível da página, como texto, imagens, vídeos e outros elementos HTML.

Tags de texto

Voltar

No HTML, as tags básicas de texto desempenham um papel crucial na estruturação e formatação do conteúdo de uma página da web. A tag <p> é usada para definir parágrafos, enquanto as tags de cabeçalho, <h1> a <h6>, são utilizadas para criar títulos de diferentes níveis, com o <h1> sendo o mais importante e <h6> o menos importante. As tags <strong> e <b> são usadas para destacar o texto em negrito, enquanto <em> e <i> são usadas para enfatizar o texto em itálico. A tag <u> é usada para sublinhar o texto, enquanto <sub> e <sup> são usadas para texto em subscrito e sobrescrito, respectivamente. Por fim, a tag <mark> é usada para destacar parte do texto com uma cor de fundo diferente, geralmente para fins de referência ou ênfase visual. Essas tags básicas desempenham um papel fundamental na criação de uma estrutura de texto clara e legível em documentos HTML.

Âncoras

Voltar

As âncoras são usadas para criar links dentro da mesma página ou para outras páginas da web. Você pode definir uma âncora usando a tag <a> com o atributo href apontando para o ID do elemento que você deseja vincular. Por exemplo, <a href="#inicio">Voltar ao início</a> irá criar um link que leva o usuário de volta ao início da página, onde o elemento possui o ID "inicio".

Citações

Voltar

As citações podem ser incorporadas ao conteúdo HTML usando as tags <q> para citações curtas e <blockquote> para citações longas. A tag <q> é usada para citações curtas que aparecem dentro de um parágrafo, enquanto a tag <blockquote> é usada para citações mais longas que geralmente são exibidas em um bloco separado, com uma margem recuada.

Inputs

Voltar

Os inputs são elementos interativos que permitem aos usuários inserir dados em um formulário. Existem vários tipos de inputs, incluindo textos, botões, caixas de seleção, checkboxes, entre outros. Cada tipo de input é representado por uma tag HTML correspondente, como <input type="text"> para entradas de texto simples, <input type="checkbox"> para checkboxes, e assim por diante.

Abreviações

Voltar

Abreviações podem ser marcadas usando a tag <abbr>. Isso permite que os navegadores mostrem a forma completa da abreviação quando o usuário passa o mouse sobre ela. Por exemplo, HTML será exibido como "HTML", mas o usuário verá a forma completa "Hypertext Markup Language" quando passar o mouse sobre ele.

Alterações

Voltar

Alterações ou exclusões podem ser indicadas usando as tags <del> para texto excluído e <ins> para texto inserido. O navegador geralmente exibirá o texto deletado com uma linha através dele e o texto inserido sublinhado. Por exemplo, texto antigo e texto novo.

Fontes

Voltar

A tag era usada para definir o estilo da fonte em versões mais antigas do HTML, mas agora é considerada obsoleta em favor do uso de CSS para estilização. No entanto, ainda é válido para fins de compatibilidade com navegadores antigos, embora seja recomendado o uso de CSS para estilização de fontes.
